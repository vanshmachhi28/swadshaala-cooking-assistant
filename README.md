SwadShaala is a modern web-based recipe management and culinary learning system, designed for users of all skill levels to cook smarter, safer, and healthier. Powered by Python & Django, with a beautiful Bootstrap UI and MySQL database, SwadShaala helps you organize recipes, scale ingredients, get utensil help, use AI tips, generate grocery lists, and more ‚Äî making cooking interactive, intelligent, and fun!

Project Overview
SwadShaala is your smart digital kitchen companion:

Add, browse, and edit recipes with images, scaling, calorie info, utensils, and step-by-step support.

Instantly generate shopping lists from selected recipes.

Get beginner help, smart tooltips, AI-powered suggestions, and ingredient substitutes.

Enjoy a smooth, intuitive, and responsive web interface.

Getting Started üöÄ
Clone this repository:

bash
git clone https://github.com/vanshmachhi28/swadshaala-cooking-assistant.git
cd swadshaala-cooking-assistant
Install the dependencies:

bash
pip install -r requirements.txt
Database & Setup:

Create and configure MySQL database and credentials in settings.py

Run Django migrations:

bash
python manage.py migrate
(Optional: Load demo data if provided.)

Run the development server:

bash
python manage.py runserver
Open:
Go to http://127.0.0.1:8000/ in your browser.

Usage
Register or log in as a new user.

Add, view, and manage recipes.

Use Smart Ingredient Scaler and Beginner Guide for help.

Add ingredients from recipes to your shopping cart.

Export shopping lists as PDF/CSV for your next grocery visit.

Admins can moderate and manage all content via the admin dashboard.
